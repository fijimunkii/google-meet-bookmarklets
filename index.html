Meet Bookmarklets

A small collection of one-click utilities for Google Meet.

Install: show the bookmarks bar (⌘⇧B / Ctrl+Shift+B) and drag any blue button below to it. Then click while in a Meet tab.

📝 Longer Gemini Notes

Switches Gemini meeting notes from Standard to Longer in one click — skips the pencil → settings → menu dance every meeting.

📝 Longer Notes
How it works · troubleshooting · source
How it works: The bookmarklet hooks window.fetch and waits for Meet to fire any RPC to MeetingSpaceService (it does this constantly during a meeting — heartbeats, presence, layout, etc.). When it sees one, it grabs the auth headers + decodes the space ID from x-goog-meeting-identifier, then immediately fires its own UpdateMeetingSpace with smart_notes_detail_level = 2. Once captured, you can experiment from console without re-clicking.

Experimenting from the console

After clicking the bookmarklet once and seeing a green toast, open DevTools console and try other enum values:

setNotesDetailLevel(1)   // Standard
setNotesDetailLevel(2)   // Longer
setNotesDetailLevel(0)   // unknown
setNotesDetailLevel(3)   // 500 — out of range

The captured headers are reused, so these calls fire instantly.

If something breaks

  • "No active meeting detected" after 15s — you're not in a meeting, or Meet's heartbeat is suppressed (rare).
  • "Couldn't update notes (HTTP …)" — the captured headers got rejected. Console shows the full response body for diagnosis.
  • Wrong meeting — if you click the bookmarklet then switch meetings, __meetCtx is stale. Re-click in the new meeting.

🌓 Invert Shared Screen


Inverts colors on the screen-share area — handy when someone's sharing a glaring white document and you want it dark, or vice versa.


🌓 Invert Share



Source

(function () {
  document.getElementsByClassName('i8wGAe')[0].style.filter = 'invert(100%)';
})();


Targets Meet's screen-share container (CSS class i8wGAe) and applies a invert(100%) CSS filter. Refreshing the page clears the inversion.









🙈 Toggle Meeting Controls


Hides (or restores) Meet's bottom controls bar — useful when presenting or focusing on the content and you don't want the chrome eating screen real estate.


🙈 Toggle Controls



Source

(function () {
  for (const c of document.querySelectorAll('[jscontroller=hVZhab]'))
    c.style.display = c.style.display === '' ? 'none' : '';
})();


Targets all elements managed by Meet's hVZhab jscontroller (the bottom controls bar) and toggles each between display:none and the default. Click the bookmarklet again to restore.
